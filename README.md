## 开发环境
- WSL2 - Ubuntu 20.04.1 Server LTS（支持16.04及以上版本），Desktop版本较为不稳定
- based on Vue, Django and Docker


## 文档
```bash
├── LICENSE
├── README.md
├── db
│   ├── backup.sh   # 备份数据库
│   └── reback.sh   # 从数据库里导回数据
├── depend.sh       # 依赖，可根据需要一键安装
├── doc.md          # 用户文档
├── docker-compose.yml
├── img
├── nodejs.sh       # nodejs安装流程
└── sec.sh          # 二次开发流程
```
